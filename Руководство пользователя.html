Butterfly v 2.0

Руководство пользователя

Общие сведения

Butterfly v 2.0 - Программа предназначена для поиска DLE сайтов по ключевым словам.
Программа использует движок Google, по этому найденные сайты будут отсортированы по Google PR по убыванию (то есть с наибольшим PR вверху).
Программа имеет эргономичный интерфейс и прячется в трей.
Все значения полей программы сохраняются и восстанавливаются при следующем открытии программы.

Как только программа находит очередной DLE сайт, она сразу же сохраняет его в файл. Так что Вы можете не беспокоится о том, если компьютер внезапно завершит работу.

Программа может искать сайты по нескольким ключевым фразам. По каждой фразе программа ищет не более 1000 сайтов. В результате общее число найденных сайтов будет не больше количества ключевых слов, умноженному на 1000.

Помните, что Google постоянно перемещает сайты по своему рейтингу и добавляет новые. Поэтому есть смысл осуществлять поиск время от времени по одним и тем же ключевым словам.

В версии 2.0

  • Новый эргономичный интерфейс;
  • Устранена остановка поиска после 600 - 900 проверок;
  • Устранено дублирование сайтов в случае встречи mysite.com и www.mysite.com;
  • Упрощена процедура поиска для удобства и лучшего понимания;
  • Возможность загрузки ключевых слов из файла.

Как искать сайты (Быстрый старт)

Для того чтобы начать поиск, сделайте:

  1. Введите в соответствующее поле ключевые фразы через запятую;
  2. Выберите исходящий файл (поле "Сохранить в"), куда будут записываться результаты;
  3. Жмите "Искать!".

Способы ввода ключевых фраз

Ключевые фразы можно вводить через запятые, и можно каждую фразу в отдельной строке. Можно комбинировать эти два варианта.

Ключевые фразы, можно вводить вручную (в специальное поле), а можно указать файл с ключевыми словами. Файл должен быть текстовый. Правила хранения ключевых фраз в файле те же (или через запятую, или в новой строке). Этот вариант удобно использовать, если у Вас есть несколько таких подготовленных файлов с ключевыми фразами, относящиеся к разным категориям.

Целевые локализации

Функция предназначена для поиска сайтов только на указанном языке и/или только из указанной страны.

Результат работы программы

Все найденные файлы сохраняются в файл, выбранный в поле "Сохранить в". Когда программа находит очередной сайт, она тут же записывает его в файл. По этому Вы можете не беспокоится о том, что компьютер выключится, и найденные сайты не сохранятся.

Каждый найденный сайт добавляется в конец файла. Если в качестве исходящего файла выбрать существующий файл и начать поиск, первоначальный файл не затрется, а новые сайты будут добавляться в конец файла как и раньше.

Вкладка "Лог"

При ошибках подключения к сети и некоторых других событиях, программа отображает сообщения и через несколько секунд скрывает их. Все эти сообщения, и некоторые другие можно увидеть в окне лога на вкладке "Лог". Окно очищается при каждом запуске программы. Кроме самого сообщения, в логах отображается дата и время сообщения.

Область статистики

В области статистики отображается текущий статус поиска:

  • Проверено - количество проверенных сайтов (по всем ключевым фразам до текущей) на то, на DLE ли движке они работают;
  • Найдено DLE - количество найденных DLE сайтов и сохраненных в файл;
  • Ошибок - количество недоступных сайтов в данный момент из проверенных.

Все эти характеристики также можно увидеть, подведя указатель мыши к иконке программы в трее.

Панель управления программой

Панель управления программой являет собой несколько кнопок "Искать!", "Продолжить", "Остановить".

  • Искать! - Начать поиск сайтов;
  • Продолжить - Продолжить поиск с данной позиции после ручной остановки поиска или в результате ошибки;
  • Остановить - Остановка поиска.

Панель статуса

Панель статуса отображает ключевую фразу, которая ищется в данный момент.
