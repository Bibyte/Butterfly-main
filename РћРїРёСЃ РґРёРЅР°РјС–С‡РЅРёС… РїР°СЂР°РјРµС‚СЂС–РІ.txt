------Параметры профиля поисковика
Description - Описание поискового профиля
SearshHost - Адрес страницы поиска у поисковика
SearshMachineBrand - Фраза, определяющая ссылку как ссылку на кеш.
LastPageCheck - фраза, определяющая страницу, которая означает, что результаты у поисковика закончились
SpamCheck - фраза, определяющая страницу, которую поисковик выдал в подозрении на спам
SearshTextPar - Параметр, содержащий поисковую фразу
CountResultsPar - Параметр, содержащий количество результатов на странице
PagePar - Параметр, содержащий текущую страницу
LangPar - Параметр, содержащий язык поиска
CountryPar - Параметр, содержащий страну поиска

------Параметры профиля поиска
LocalPHPCheckURL - Локальный дарес страницы для проверки на PHP
ParamsForPHPCheck - Параметры для проверки на PHP
CheckForPHP - Флаг проверки на PHP
WhiteListSize - Размер белого списка
WhiteRecordSize[x] - Размер первой записи (y)
WhiteRecord[xk] - Запись белого списка
BlackListSize - Размер черного списка
BlackRecordSize[x] - Размер первой записи (y)
BlackRecord[xk] - Запись черного списка
